# Wind Farm Investment Model (Excel)

## Overview
This repository contains a group university coursework project involving the development of an Excel-based financial model to assess the profitability of a wind farm investment under different scenarios.

## Project Objective
The aim of the model was to analyse how changes in key assumptions affect the profitability of a wind farm project. These assumptions included factors such as turbine size, wind speed, location, electricity prices, borrowing and lending rates, lease costs, and subsidies.

## What the Model Does
The spreadsheet allows users to vary investment assumptions and evaluate the financial performance of the project using balance sheet outputs and profitability measures.
